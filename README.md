 🧮 Ajiri Calculator PWA

A simple, responsive, and installable calculator web app built using HTML, CSS, and JavaScript.  
This project is also a **Progressive Web App (PWA)**, meaning it can be installed on your phone like a real app and works offline.

---

🚀 Live Demo

https://ajiri-calculator.netlify.app/
---

✨ Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clean and modern UI design
- Fully responsive (works on mobile & desktop)
- Installable as a PWA (Add to Home Screen)
- Offline support using Service Worker
- Fast performance

---

📱 PWA Features
- Can be installed like a native app
- Works without internet after first load
- App-like fullscreen experience
- Custom app icon (192px & 512px)

---

🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PWA (Manifest + Service Worker)

⚙️ How It Works
1. User clicks number buttons
2. Expression is built dynamically
3. JavaScript evaluates the expression
4. Result is displayed instantly
5. Service worker caches files for offline use

---

📦 Installation (for users)
1. Open the app link in Chrome
2. Click "Add to Home Screen"
3. Install it like a real mobile app
